„xhtmldbg“ ist ein Webseiten Debugger für Webentwickler und Konsumenten.

Der Grundgedanke dieses Projektes ist einen Webdebugger zu Entwickeln der Gleichzeitig den Quelltext als auch die damit zusammenhängenden Protokolle erkennt und auf ihre Validität prüft.

„xhtmldbg“ bietet neben dem prüfen auf einen Validen Quelltext zusätzliche Optionen:

  • Überwachen des HTTP Datenkopfes,
  • überwachen von abgesendeten Post Variablen aus einem Formular heraus,
  • CSS StyleSheet Überprüfung,
  • einen Dom Inspektor zur besseren Darstellung der Dokumenten Struktur,
  • anzeigen von CSS StyleSheet Prädikaten,
  • anzeige der aktuell verwendeten Cookies,
  • ausgeben von JavaScript Fehlern,
  • automatisches Aktualisieren in Zehner schritten bis 60 Sekunden,
  • SEO Schlüsselwort Überprüfung,
  • schnelles Umschalten des USER-AGENT,
  • strikte Einhaltung des RFC 2109 (HTTP State Management Mechanism) für die Cookie Behandlung,
  • ATOM/RSS/RDF Nachrichten Dokumente Validieren,
  • und natürlich das Validieren und Bereinigen des Markup Quelltextes.

„xhtmldbg“ erscheint unter der GPLv3.
